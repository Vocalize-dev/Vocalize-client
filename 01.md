# Bot

接続時：
  POST http://server/internal/connect
    {
      "type": "connect",
    }
    {
      "state_id": "st4t3v0",
    }
    ユーザーに表示：
      接続IDは「st4t3v0」です。

切断：
  POST http://server/internal/connect
    {}

# Client

接続時：
  POST http://server/api/connect/st4t3v0
    {
      "type": "connect",
      "state_id": "st4t3v0",
    }
    {
      "code": "ok",
    }
